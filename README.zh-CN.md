# HTML Studio

[English](README.md) | [中文](README.zh-CN.md)

一款轻量级浏览器端 HTML 编辑器，支持源码编辑、可视化编辑和实时预览。

## 功能特性

- 源码模式：支持 HTML 语法高亮、行号、标签匹配、代码片段插入、格式化和代码折叠。
- 可视化模式：像 Word 一样直接编辑内容，不需要看到 HTML 标签。
- 实时预览：使用 iframe 实时渲染，支持桌面、平板、手机视图切换。
- 可拖动分栏：编辑区和预览区之间的比例可以自由调整。
- 文件操作：支持新建、打开、保存、导出和自动保存。
- 主题切换：支持浅色和深色主题。
- 专注模式：隐藏干扰元素，提供更纯粹的编辑体验。
- 撤销功能：支持 `Ctrl+Z`。

## 使用方式

直接在浏览器中打开 `index.html`。

无需构建步骤，也不需要启动开发服务器。

## 快捷键

- `Ctrl+S`：保存
- `Ctrl+O`：打开
- `Ctrl+N`：新建文档
- `Ctrl+Z`：撤销
- `Ctrl+Shift+F`：格式化 HTML
- `F11`：切换专注模式
- `Esc`：退出专注模式

## 文件说明

- `index.html`：应用页面结构
- `styles.css`：布局、主题、编辑器和预览样式
- `app.js`：编辑器行为、实时预览、文件操作、历史记录和模式切换逻辑

## 说明

编辑器完全运行在浏览器中。自动保存内容存储在 `localStorage`；保存和导出会通过浏览器下载文件，或在浏览器支持时使用文件选择器直接写入本地文件。
